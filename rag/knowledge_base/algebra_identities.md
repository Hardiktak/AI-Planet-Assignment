# Algebra — Identities and Expansions

## Basic Algebraic Identities

1. **(a + b)² = a² + 2ab + b²**
2. **(a − b)² = a² − 2ab + b²**
3. **(a + b)(a − b) = a² − b²**
4. **(a + b)³ = a³ + 3a²b + 3ab² + b³**
5. **(a − b)³ = a³ − 3a²b + 3ab² − b³**
6. **a³ + b³ = (a + b)(a² − ab + b²)**
7. **a³ − b³ = (a − b)(a² + ab + b²)**

## Multinomial Expansions

8. **(a + b + c)² = a² + b² + c² + 2ab + 2bc + 2ca**
9. **a³ + b³ + c³ − 3abc = (a + b + c)(a² + b² + c² − ab − bc − ca)**
   - Special case: if a + b + c = 0, then a³ + b³ + c³ = 3abc

## Binomial Theorem

- **(x + y)ⁿ = Σ (n choose k) · x^(n−k) · y^k** for k = 0 to n
- General term: T(r+1) = C(n, r) · x^(n−r) · y^r
- Number of terms in expansion of (x + y)ⁿ = n + 1
- Middle term: if n is even → T(n/2 + 1); if n is odd → T((n+1)/2) and T((n+3)/2)

## Sophie Germain Identity

- **a⁴ + 4b⁴ = (a² + 2b² + 2ab)(a² + 2b² − 2ab)**

## JEE Application Tips

- Factor before expanding — saves time on MCQ problems.
- When you see a³ + b³ + c³, always check if a + b + c = 0.
- The binomial coefficient C(n, r) = C(n, n−r) — use the smaller value for faster computation.
