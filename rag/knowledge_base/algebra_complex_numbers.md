# Algebra — Complex Numbers

## Definition

A complex number z = a + bi, where a = Re(z), b = Im(z), i = √(−1).

## Basic Operations

- **Addition**: (a + bi) + (c + di) = (a+c) + (b+d)i
- **Multiplication**: (a + bi)(c + di) = (ac − bd) + (ad + bc)i
- **Conjugate**: z̄ = a − bi
- **Modulus**: |z| = √(a² + b²)
- **z · z̄ = |z|²** = a² + b²

## Polar Form

z = r(cos θ + i sin θ) = r · e^(iθ), where r = |z|, θ = arg(z)

- **Multiplication in polar**: z₁z₂ = r₁r₂ · e^(i(θ₁+θ₂))
- **Division**: z₁/z₂ = (r₁/r₂) · e^(i(θ₁−θ₂))

## De Moivre's Theorem

(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)

## Euler's Formula

e^(iθ) = cos θ + i sin θ

## Roots of Unity

The nth roots of unity are: e^(2πik/n) for k = 0, 1, ..., n−1
- They form a regular n-gon on the unit circle
- Sum of all nth roots of unity = 0
- Product of all nth roots of unity = (−1)^(n+1)

## Properties

- |z₁ + z₂| ≤ |z₁| + |z₂| (Triangle inequality)
- |z₁ · z₂| = |z₁| · |z₂|
- arg(z₁z₂) = arg(z₁) + arg(z₂)
- If |z| = 1, then z̄ = 1/z

## JEE Tips

- For locus problems, write z = x + iy and convert to Cartesian.
- |z − z₁| = |z − z₂| represents the perpendicular bisector of z₁z₂.
- |z − z₁| = r represents a circle with center z₁ and radius r.
- arg(z − z₁) = θ represents a ray from z₁ at angle θ.
