# Solution Templates — Step-by-Step Strategies

## Template 1: Solving Quadratic Equations

1. Write in standard form: ax² + bx + c = 0
2. Compute discriminant: D = b² − 4ac
3. Determine nature of roots (D > 0, = 0, < 0)
4. Apply quadratic formula: x = (−b ± √D) / 2a
5. Simplify and verify by substitution

## Template 2: Finding Derivatives

1. Identify the function type (polynomial, trig, exponential, composite)
2. Determine which rules apply (power, chain, product, quotient)
3. Apply rules step by step
4. Simplify the result
5. Verify: check at a known point or use dimensional consistency

## Template 3: Evaluating Limits

1. Try direct substitution
2. If indeterminate (0/0, ∞/∞), identify the form
3. Choose technique: factoring, rationalization, standard limits, L'Hôpital
4. Apply the technique
5. Evaluate the simplified expression
6. State the final answer

## Template 4: Optimization Problems

1. Define the variable to optimize
2. Express it as a function of one variable
3. Determine the domain/constraints
4. Find critical points: set f'(x) = 0
5. Use second derivative test or boundary analysis
6. State the maximum/minimum value and where it occurs

## Template 5: Probability Problems

1. Define the sample space
2. Identify the event of interest
3. Choose the approach: direct counting, complement, conditional
4. Calculate the probability
5. Verify: 0 ≤ P ≤ 1 and the answer makes intuitive sense

## Template 6: Matrix/Determinant Problems

1. Write the system in matrix form AX = B
2. Compute det(A)
3. If det(A) ≠ 0: use Cramer's rule or A⁻¹
4. If det(A) = 0: check for consistency (adj(A)·B = 0?)
5. State the solution or nature of the system

## Template 7: Integration Problems

1. Identify the integral type
2. Choose technique: direct formula, substitution, parts, partial fractions
3. Apply the technique step by step
4. For definite integrals: apply limits
5. Simplify and verify by differentiating the result

## General Problem-Solving Strategy

1. **Read carefully**: identify what is given and what is asked
2. **Draw a diagram** if applicable
3. **Choose a method**: formula, substitution, or first principles
4. **Execute** step by step — show intermediate work
5. **Verify**: substitute back, check dimensions, sanity-check the answer
6. **State the answer** clearly
