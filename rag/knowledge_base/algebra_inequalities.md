# Algebra — Inequalities and Ranges

## Basic Inequality Rules

1. If a > b and c > 0, then ac > bc
2. If a > b and c < 0, then ac < bc (inequality flips)
3. If a > b > 0, then 1/a < 1/b
4. |a + b| ≤ |a| + |b| (Triangle Inequality)
5. |a − b| ≥ ||a| − |b||

## AM-GM Inequality

For positive real numbers a₁, a₂, ..., aₙ:

**AM ≥ GM**: (a₁ + a₂ + ... + aₙ)/n ≥ (a₁ · a₂ · ... · aₙ)^(1/n)

Equality holds when a₁ = a₂ = ... = aₙ.

### Common Applications
- For two numbers: (a + b)/2 ≥ √(ab)
- Minimum of a + 1/a for a > 0 is 2 (when a = 1)
- Minimum of a² + b² given a + b = constant: use AM-GM or Cauchy-Schwarz

## Cauchy-Schwarz Inequality

(a₁² + a₂² + ... + aₙ²)(b₁² + b₂² + ... + bₙ²) ≥ (a₁b₁ + a₂b₂ + ... + aₙbₙ)²

## Quadratic Inequality

To solve ax² + bx + c > 0 (or < 0):
1. Find roots using quadratic formula
2. Draw sign diagram on number line
3. If a > 0, the parabola opens upward (positive outside roots)
4. If a < 0, the parabola opens downward (positive between roots)

## Wavy Curve Method

For rational inequalities like (x−a)(x−b)/((x−c)(x−d)) > 0:
1. Find all zeros and undefined points
2. Plot on number line
3. Start from +∞ on the right, alternate signs at each simple root
4. Repeated roots don't change sign

## JEE Tips

- AM-GM is the most tested inequality in JEE — practice recognizing when to apply it.
- When finding range of a function, set f(x) = y and solve for x; the values of y for which x is real give the range.
- For |f(x)| < k, solve −k < f(x) < k.
